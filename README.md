紹介
-------------

Jupiterは無料のMinecraft Pocket Edition用のサーバーソフトウェアです。
大部分のソースはNukkitのものを使用しています。

* Javaで書かれているから速い
* Javaだから多くのプラットフォームに対応しています。
* Nukkitプラグインが使えます。

--------------------

簡単です。
